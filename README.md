# PySTEM Institute Website

This is a simple GitHub Pages-ready static website for PySTEM Institute.

## Files

- `index.html` — Homepage with hero section, course cards, why-us section, free trial CTA, and contact section.
- `courses.html` — Detailed course sections for Physics, Chemistry, Mathematics, Python, and AI/ML.
- `styles.css` — All styling.
- `script.js` — Mobile navigation and automatic footer year.

## Why this setup?

This website uses plain HTML, CSS, and vanilla JavaScript because it is:

- Fast to load
- Free to host on GitHub Pages
- Easy to update
- Search-engine friendly
- Future-compatible with Google Forms, Streamlit demos, chatbot widgets, and QR-code campaigns

2. `(416) 555-0123`
3. `info@pystem.ca`
4. `www.pystem.ca`

## Suggested deployment on GitHub Pages

1. Create a GitHub account.
2. Create a public repository, for example: `pystem-website`.
3. Upload all files into the repository root.
4. Go to `Settings` → `Pages`.
5. Choose source branch: `main`.
6. Choose folder: `/root`.
7. Save.
8. GitHub will give you a public website URL.

## QR Code

Once the website URL is ready, create a QR code that points to the homepage.

Recommended flow:

QR code → Website homepage → Registration button → Google Form → Google Sheets

Do not point the QR code directly to the form unless you only want a registration-only campaign.
