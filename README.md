# PySTEM Institute Website

This is the integrated PySTEM website version using PySTEM_v0 as the base and selected PySTEM_v1 improvements.

## Files

- `index.html` — Homepage with Home, Courses, Learning Plans, Resources, Teaching Plans, FAQs and Contact sections.
- `courses.html` — Detailed course roadmaps for Physics, Chemistry, Mathematics, Python and AI/ML.
- `styles.css` — Dark STEM visual design, responsive cards, plan layouts and FAQ styling.
- `script.js` — Mobile menu, current year and FAQ accordion.
- `images/logo.png` — Keep your existing logo file at this exact path.
