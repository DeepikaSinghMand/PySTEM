Course Sections

Ontario-aligned STEM tutoring and future-ready computational learning

These course pages are written so that you can easily update topics, grade levels, prerequisites, delivery mode and registration links.

01

Physics

Grades 7–12

Core Topics

  • Mechanics, motion, forces and energy
  • Kinematics and problem-solving strategies
  • Electricity, circuits and magnetism
  • Thermodynamics, optics and waves

Learning Outcomes

  • Develop clear physical intuition
  • Translate word problems into equations
  • Improve unit analysis and graph interpretation
  • Prepare for tests, exams and advanced STEM pathways
02

Chemistry

Grades 7–12

Core Topics

  • Atoms, bonding and periodic trends
  • Stoichiometry and chemical reactions
  • Organic and inorganic chemistry foundations
  • Biochemistry, lab skills and safety

Learning Outcomes

  • Build strong conceptual understanding
  • Master calculation-based chemistry problems
  • Connect chemical principles to real-world examples
  • Prepare for school assessments and science enrichment
03

Mathematics

Grades K–12

Core Topics

  • Arithmetic, algebra and functions
  • Trigonometry and geometry
  • Calculus foundations and advanced problem solving
  • Statistics, probability, matrices and vectors

Learning Outcomes

  • Strengthen fundamentals from grade level upward
  • Improve symbolic reasoning and mathematical fluency
  • Develop exam-ready problem-solving techniques
  • Prepare for physics, coding and machine learning pathways
04

Python Programming

Grades 6–12

Core Topics

  • Variables, loops, conditionals and functions
  • Lists, dictionaries and data structures
  • File handling, plotting and debugging
  • NumPy, Pandas and real-world mini projects

Learning Outcomes

  • Write clean beginner-to-intermediate Python code
  • Use programming for math and science problems
  • Build small portfolio projects
  • Prepare for AI, ML and data science learning
05

AI & Machine Learning

Grades 9–12

Core Topics

  • Data science foundations and model thinking
  • Regression, classification and validation
  • Neural networks and deep learning concepts
  • Mathematics behind learning: functions, derivatives and optimization

Learning Outcomes

  • Understand how machine learning models learn from data
  • Connect calculus and algebra to model training
  • Build beginner-friendly AI/ML projects in Python
  • Prepare for advanced STEM, AI and research-oriented learning
